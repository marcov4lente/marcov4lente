
As a proof of concept I decided to build a quick [Python web crawler](https://github.com/marcov4lente/google-play-data-scanner) to analyse the Google Play Store's top free applications. I wrote this webcrawling using the Beautiful Soap package, which essentially allows one to parse html and select and anipulate it's elements. Beautiful Soap made it incredibly simple to crawl the Google Play site and harvest it's data.

I set the crawler to run daily at 23:59:00 for a week to collect some data.
